# fanuc_m710ic_description

This package contains a URDF description for the Fanuc M-710iC manipulator.

## Overview

The original Fanuc M-710iC description is published in Xacro under the BSD 3-Clause license in the [fanuc](https://github.com/ros-industrial/fanuc) repository, which is part of the [ROS-Industrial](http://wiki.ros.org/Industrial) program. It was converted to URDF by [St√©phane Ancelot](https://github.com/sancelot). See the main [fanuc](https://wiki.ros.org/fanuc) page on the ROS wiki for more information on usage.

## Reporting issues

This description is distributed as-is in the hope it can be useful. If you face an issue with the URDF file itself, open an issue here. Please report any other issue to the upstream repository.
